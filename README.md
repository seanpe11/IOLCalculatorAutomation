Browser automation to make it easier to write into the IOL calculator provided by [ECRS](https://iolcalculator.escrs.org/). This is NOT a fully featured IOLS calculator; the only intention is to make it easier for clinical researchers to bring their excel sheets and easily automate the process of making the calculations, instead of manually having to type out each calculation. Please also note that the ESCRS website requires a captcha; after the autofill proceeds for the line in your excel sheet, you will still have to manually complete the CAPTCHA.
